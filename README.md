# Road Lane Detection with VGG-UNet on TuSimple Dataset | Image Segmentation Tutorial

# Project Information

In this project, we explore road lane detection using VGG-UNet on the TuSimple dataset. This deep learning approach applies image segmentation to accurately identify lanes in real-world driving scenarios. We cover data preprocessing, model training, and result visualization, providing insights into how deep learning enhances autonomous driving systems.

The TuSimple dataset consists of 6,408 road images on US highways. The resolution of the image is 1280×720. The dataset is composed of 3,626 for training, 358 for validation, and 2,782 for testing called the TuSimple test set of which the images are under different weather conditions.

**Download link:** https://www.kaggle.com/datasets/manideep1108/tusimple
